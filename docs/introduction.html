VS Toolkit the mobile HTML5 toolkit used with VSD

This toolkit was created to easily develop Hybrid Applications for mobile devices (for instance on top of PhoneGap)

The main features are:

  • Objects oriented toolkit
  • Framework driven by a component approach
  • Optimized for mobile devices
    • Small memory footprint
    • Support of memory deallocation (image, bindings, object...)
    • Performance (JS and Rendering)

The IDE ViniSketch Designer (VSD) use this toolkit for creating application.
