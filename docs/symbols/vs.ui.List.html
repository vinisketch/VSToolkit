Class vs.ui.List

Extends vs.ui.AbstractList.

The vs.ui.List class draw a list of ListItem and allows the user to select one object from it.

Events:

  • itemselect, fired when a item is selected. Event Data = {index, item data}

To reduce performance issues, you can deactivate events handling for the list, using vs.ui.List#itemsSelectable property. Data can be filtered. The filter he array contains the member to filters and filter:
